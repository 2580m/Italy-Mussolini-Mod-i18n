# Italy-Mussolini-Mod-i18n
这是意大利曾短暂到达过未来Mod的翻译仓库

流程：
fork一份到你的账号下
将你的翻译添加到你fork的仓库中，请维持路径和文件名以及编码的标准化，例如英文本地化要放到 localisation/english 下，文件名是形如 xxxx_l_english.yml 的格式，文件编码为 UTF-8 with BOM.
在你的仓库中点击上方的 Pull Request 按钮，创建一个拉取请求。
等待你的拉取请求被合并或者被要求修改。

Usage:

Creat a Fork
Add your translations to forked repository, plz keep the dir, file name and file coder for standard, example dir: localisation/english/xxx_english.yml, example file name: xxxx_l_english.yml, force coder: UTF-8 with BOM.
Create a PR
Waiting Your PR to be accepted or modify your PR
